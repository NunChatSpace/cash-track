# Rules (Generic)

## Required Conventions

1) Progress logs live in `.claude/progress/`
   - Name format: `YYYY-MM-DD-HH-mm-short-title.md`

2) Commit messages
   - Prefer: `feat: ...`, `fix: ...`, `chore: ...`

3) Data scoping
   - Scope data access (e.g., user/org/tenant)
   - Define defaults explicitly

4) Status handling
   - Use explicit status fields for workflows
   - Avoid implicit state
